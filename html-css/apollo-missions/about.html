About This Project

Purpose

This website was created to explore and showcase NASA’s Apollo program, which played a major role in human space exploration and the first Moon landing.

What I Learned

  • How to structure a multi-page website using HTML
  • How to link pages together using anchor tags
  • How to organize information in a clear and readable way
  • How to present historical and scientific content in a web format
  • How to design a user-friendly interface

Why the Apollo Program

The Apollo missions represent one of the most important achievements in human history. I chose this topic because I am interested in space exploration and future careers in engineering and aerospace, including opportunities like NASA internships.
